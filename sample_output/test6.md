# main

The top-level namespace.



* * *

### main.init() 

Initializes everything.



### main.dispose() 

Disposes everything.



### main.do(thing) 

Have a Worker do some Thing.  See main.Thing.

**Parameters**

**thing**: `main.Thing`, The Thing to do.  See main.Thing.




# util

Namespace for utility functions.



* * *

### util.foo() 

Run the Foo utility.



### util.bar() 

Run the Bar utility.
